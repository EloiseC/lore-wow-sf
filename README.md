# lore-wow-sf
Project about Lore of WoW in Symfony

First, you must to install this on your computer :
- node
- yarn
- composer

Then, run this cmd :
- composer install
- yarn install
- yarn build
- yarn watch

About bdd, first, config your .env then run :
- php bin/console d:d:c
- php bin/console d:s:u --f
